监控引擎

深度监控引擎详解

GEMMB 内置多种专业监控引擎，每一种都经过大规模生产环境验证，提供细粒度的数据采集与分析能力。

SNMP 监控引擎

高度可配置的 SNMP 轮询引擎，厂商无关设计，可监控任何支持 SNMP 的设备。

  • 状态、统计、阈值三维监控
  • 灵活实体模型，自动发现设备层级与依赖关系
  • 内置远程排障工具：SNMP 实时查询、Ping、Trace Route
  • 详细数据分析导航树，支持"忙时分析"
  • 支持 SNMP v1/v2c/v3，Trap 告警接收

HTTP 监控引擎

网站 / Web 服务 / Web 应用的 HTTP 协议健康检查与性能监控。

  • 可定制脚本引擎，轮询多种内容类型
  • HTTP 响应状态与性能统计（总响应时间、内容类型分解）
  • SSL 证书监控：过期提醒、名称不匹配告警、过时协议检测
  • 支持 HTTP/HTTPS 协议，可配置请求头和认证信息

DNS 监控引擎

全面 DNS 解析服务监控，支持所有主流 DNS 记录类型。

  • 支持 A、MX、NS、URI、CNAME 等所有主流记录类型
  • 查询结果与预配置值自动比对，异常即触发告警
  • DNS 组管理，支持动态 IP 地址设备

NetFlow / sFlow 流量分析

多维度网络流量可视化分析，支持 NetFlow v5/v9 和 sFlow 协议。

  • Top Talkers、Top Listeners、Top Conversations
  • Top Applications、Top Protocols、Top TOS
  • NetFlow Relay 中继功能，数据转发至第三方系统

Syslog 日志管理

分布式 Poller 高密采集 Syslog，支持全文检索与定制化过滤。

  • 基于 Syslog 自动生成告警
  • 日志采集存储与全文检索
  • C# 脚本定制化过滤规则
Syslog Relay 中继

支持按客户/设备精细化转发控制，加密转发至合规冷备存储区，满足等保 2.0 规定的 180 天原始日志存储要求。

Smart Monitoring 智能监控

智能降低工单数量，有效过滤偶发性抖动告警，减少无效工单。

1
异常进入"观察"状态

非正常告警不立即创建工单，而是进入观察期

2
观察期内自动判断

恢复正常 → 自动关闭；超时未恢复 → 按状态设定严重等级

3
有效过滤噪声

偶发性抖动告警被过滤，只有持续性故障才创建工单

Configure Harvest 配置采集

定时采集网络设备配置版本，自动比对变更差异，发现未授权变更时主动报警。

灵活脚本引擎

支持通过 Telnet、SSH 或 SNMP+TFTP 采集设备配置。

历史版本比对

配置文档自动保存并与历史版本逐行比对，清晰展示变更差异。

定时调度执行

可按日定时调度执行，防止配置漂移，确保合规基线。

网络拓扑发现与根因分析

自动发现被管设备的轮询拓扑与依赖关系，动态拓扑可视化展示。集成服务台进行根因分析：多台设备不可达时，自动关联为同一事件生成一张工单，避免告警风暴。

分布式采集

分布式 Poller 架构

借助高可扩展的分布式监控机制，GEMMB 在客户多区域或多分支网络中统一部署 Poller 代理端，支持多租户与高安全加密通信。

多租户 VRF 隔离

每个 Poller 支持多租户部署，不同客户/部门之间的监控数据在网络层面绝对隔离。

SSL 加密通信

Poller 与中心平台之间的所有通信均通过 SSL/TLS 加密，防止数据在传输过程中被截获。

弹性拓扑扩展

从微型分支到跨国云环境无缝扩展，Master Poller 架构支持全球分布式部署，单虚拟 Poller 支持多客户。

远程集中管理

支持远程集中管理与配置下发，无需在每个节点安排运维人员。内置看门狗自动恢复软件故障。

Poller 部署与配置能力

支持 Windows、Linux 和 Docker 部署
按客户、区域、位置或设备级别灵活分配
基于角色的 Poller 管理，支持委托部署
可配置 Ping 静默时间、SNMP 轮询间隔
HTTP 轮询间隔、协议端口可配置
统计汇总周期、内置看门狗机制
服务器管理

服务器 Agent 深度管理

通过轻量级 Windows / Linux Agent 实现服务器环境的深度监控与远程管理，覆盖从指标采集到远程桌面、事件日志到脚本执行的完整管理场景。

Windows / Linux Agent

轻量级部署（约 100MB 内存，极低 CPU 占用），通过 Web 门户集中配置。

  • 灵活脚本化监控引擎
  • 状态/统计/阈值监控与告警生成
  • 全局默认配置 + 单项覆盖
  • 自定义时间段阈值

Windows 远程桌面

即时（Just-in-Time）远程桌面访问，安全与便捷的完美结合。

  • 连接前自动加入远程桌面用户组
  • 连接结束后自动移除权限
  • 可选加入本地管理员组
  • 与密码保险箱集成

Windows 桌面共享

通过 Server Agent 作为主机实现安全桌面共享，支持 Connector 客户端。

  • 仅查看或交互控制模式
  • 自定义访问权限级别
  • 支持安全桌面（UAC）查看与交互

Windows 事件日志监控

灵活选择监控的事件日志和事件级别，精细化过滤。

  • 事件 ID 过滤（包含/排除/范围）
  • 事件汇总：同窗口内相同事件合并计数
  • 事件过滤脚本，自定义告警文本

Shell 脚本与系统更新

直接在 Web 端执行 Shell / PowerShell 脚本，无需 SSH 登录。

  • 在线执行 Shell / PowerShell 脚本
  • Windows Update 过期更新告警
  • 自动安装更新 / 手动检查下载

自定义 KPI 看板

拖拽式仪表盘，支持多维度数据可视化。

  • 按角色定制监控视图
  • CIO 概览 / 工程师详情
  • SLA 达标率自动追踪
