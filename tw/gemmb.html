監控引擎

深度監控引擎詳解

GEMMB 內置多種專業監控引擎，每一種都經過大規模生產環境驗證，提供細粒度的數據採集與分析能力。

SNMP 監控引擎

高度可配置的 SNMP 輪詢引擎，廠商無關設計，可監控任何支持 SNMP 的設備。

  • 狀態、統計、閾值三維監控
  • 靈活實體模型，自動發現設備層級與依賴關係
  • 內置遠程排障工具：SNMP 實時查詢、Ping、Trace Route
  • 詳細數據分析導航樹，支持"忙時分析"
  • 支持 SNMP v1/v2c/v3，Trap 告警接收

HTTP 監控引擎

網站 / Web 服務 / Web 應用的 HTTP 協議健康檢查與性能監控。

  • 可定製腳本引擎，輪詢多種內容類型
  • HTTP 響應狀態與性能統計（總響應時間、內容類型分解）
  • SSL 證書監控：過期提醒、名稱不匹配告警、過時協議檢測
  • 支持 HTTP/HTTPS 協議，可配置請求頭和認證信息

DNS 監控引擎

全面 DNS 解析服務監控，支持所有主流 DNS 記錄類型。

  • 支持 A、MX、NS、URI、CNAME 等所有主流記錄類型
  • 查詢結果與預配置值自動比對，異常即觸發告警
  • DNS 組管理，支持動態 IP 地址設備

NetFlow / sFlow 流量分析

多維度網絡流量可視化分析，支持 NetFlow v5/v9 和 sFlow 協議。

  • Top Talkers、Top Listeners、Top Conversations
  • Top Applications、Top Protocols、Top TOS
  • NetFlow Relay 中繼功能，數據轉發至第三方系統

Syslog 日誌管理

分佈式 Poller 高密採集 Syslog，支持全文檢索與定製化過濾。

  • 基於 Syslog 自動生成告警
  • 日誌採集存儲與全文檢索
  • C# 腳本定製化過濾規則
Syslog Relay 中繼

支持按客戶/設備精細化轉發控制，加密轉發至合規冷備存儲區，滿足等保 2.0 規定的 180 天原始日誌存儲要求。

Smart Monitoring 智能監控

智能降低工單數量，有效過濾偶發性抖動告警，減少無效工單。

1
異常進入"觀察"狀態

非正常告警不立即創建工單，而是進入觀察期

2
觀察期內自動判斷

恢復正常 → 自動關閉；超時未恢復 → 按狀態設定嚴重等級

3
有效過濾噪聲

偶發性抖動告警被過濾，只有持續性故障才創建工單

Configure Harvest 配置採集

定時採集網絡設備配置版本，自動比對變更差異，發現未授權變更時主動報警。

靈活腳本引擎

支持通過 Telnet、SSH 或 SNMP+TFTP 採集設備配置。

歷史版本比對

配置文檔自動保存並與歷史版本逐行比對，清晰展示變更差異。

定時調度執行

可按日定時調度執行，防止配置漂移，確保合規基線。

網絡拓撲發現與根因分析

自動發現被管設備的輪詢拓撲與依賴關係，動態拓撲可視化展示。集成服務檯進行根因分析：多臺設備不可達時，自動關聯爲同一事件生成一張工單，避免告警風暴。

分佈式採集

分佈式 Poller 架構

藉助高可擴展的分佈式監控機制，GEMMB 在客戶多區域或多分支網絡中統一部署 Poller 代理端，支持多租戶與高安全加密通信。

多租戶 VRF 隔離

每個 Poller 支持多租戶部署，不同客戶/部門之間的監控數據在網絡層面絕對隔離。

SSL 加密通信

Poller 與中心平臺之間的所有通信均通過 SSL/TLS 加密，防止數據在傳輸過程中被截獲。

彈性拓撲擴展

從微型分支到跨國雲環境無縫擴展，Master Poller 架構支持全球分佈式部署，單虛擬 Poller 支持多客戶。

遠程集中管理

支持遠程集中管理與配置下發，無需在每個節點安排運維人員。內置看門狗自動恢復軟件故障。

Poller 部署與配置能力

支持 Windows、Linux 和 Docker 部署
按客戶、區域、位置或設備級別靈活分配
基於角色的 Poller 管理，支持委託部署
可配置 Ping 靜默時間、SNMP 輪詢間隔
HTTP 輪詢間隔、協議端口可配置
統計彙總週期、內置看門狗機制

服務器 Agent 深度管理

通過輕量級 Windows / Linux Agent 實現服務器環境的深度監控與遠程管理，覆蓋從指標採集到遠程桌面、事件日誌到腳本執行的完整管理場景。

Windows / Linux Agent

輕量級部署（約 100MB 內存，極低 CPU 佔用），通過 Web 門戶集中配置。

  • 靈活腳本化監控引擎
  • 狀態/統計/閾值監控與告警生成
  • 全局默認配置 + 單項覆蓋
  • 自定義時間段閾值

Windows 遠程桌面

即時（Just-in-Time）遠程桌面訪問，安全與便捷的完美結合。

  • 連接前自動加入遠程桌面用戶組
  • 連接結束後自動移除權限
  • 可選加入本地管理員組
  • 與密碼保險箱集成

Windows 桌面共享

通過 Server Agent 作爲主機實現安全桌面共享，支持 Connector 客戶端。

  • 僅查看或交互控制模式
  • 自定義訪問權限級別
  • 支持安全桌面（UAC）查看與交互

Windows 事件日誌監控

靈活選擇監控的事件日誌和事件級別，精細化過濾。

  • 事件 ID 過濾（包含/排除/範圍）
  • 事件彙總：同窗口內相同事件合併計數
  • 事件過濾腳本，自定義告警文本

Shell 腳本與系統更新

直接在 Web 端執行 Shell / PowerShell 腳本，無需 SSH 登錄。
